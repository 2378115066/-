# Linux文件特殊属性

## chattr命令

```
chattr命令用于设置文件的特殊属性，格式为“chattr［参数］文件”。


如果想要把某个特殊属性添加到文件上，则需要在命令后面追加“+参数”
如果想要把某个特殊属性移出文件上，则需要追加“-参数”

格式：chattr选项 文件或目录
	i：不能更改
	a：不能删除和修改，只能追加


```

## Isattr命令

```
Isattr命令用于显示文件的特殊属性，在Linux系统中，文件的特殊属性必须使用Isattr命令来查看，Is之类的命令看不出来，一旦使用Isattr命令后，文件上被赋予的特殊属性马上就会原形毕露。此时可以按照显示的隐藏权限的类型（字母），使用chattr命令将其去掉

格式：lsattr 文件或目录
出社
国家信息安全水平考试8
NATIONAL INFORMATION SECURITY TEST PROGRAM 十
单击此处添加备注 幻灯片8／10
默认设计模板 
```

## umask

```
umask值用于设置用户在创建文件时的默认权限，当我们在系统中创建目录或文件时，目录或文件所具有的默认权限就是由umask值决定的。

对于root用户，系统默认的umask值是0022；对于普通用户，系统默认的umask值是0002。

执行umask命令可以查看当前用户的umask值

默认情况下，对于目录，用户所能拥有的最大权限是777；对于文件，用户所能拥有的最大权限是目录的最大权限去掉执行权限，即666。因为x执行权限对于目录是必须的，没有执行权限就无法进入目录，而对于文件则不必默认赋予x执行权限。

计算方式：
	对于目录：777—umask＝创建的目录的默认权限
	对于文件：666—umask＝创建的文件的默认权限-
```
