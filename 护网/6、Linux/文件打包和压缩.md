## 文件的打包与压缩

```
打包：将一大堆文件或目录什么的变成一个总的文件

压缩：将一个大的文件通过一些压缩算法变成一个小文件
```

```
为什么要区分这两个概念呢？其实这源于Linux中的很多压缩程序，只能针对一个文件进行压缩，这样当你想要压缩一大堆文件时，你就得先借助另外的工具将这一大堆文件先打成一个包，然后再用压缩程序进行压缩。
```



## tar命令：

```
用来压缩和解压文件。tar本身不具有压缩功能。他是调用压缩功能实现的

-c：打包文件
-x：从包中提取文件
-t：查看内容 
-f：指定文件
-v；显示执行过程
—z：使用gzip压缩或者解压

tar cf file.tar files          #创建包含files的tar文件
tar xf files.tar               #从files.tar中提取文件 8
tar czf file.tar.gz files      #使用Gzip压缩创建tar文件
tar xzf file.tar.gz            #使用Gzip提取tar文件
```
