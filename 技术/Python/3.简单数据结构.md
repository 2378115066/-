# 二、判断语句

## 1.定义：

```
如果某些条件满足，才能做某件事情，而不满足时不允许做，这就是所谓的判断
不仅生活中有，在软件开发中“判断”功能也经常会用到
```

## 2.语法：

```python
if 要判断的条件:
    执行语句

if 条件:
	执行语句1
else:
    执行语句2
    
if 条件1:
   执行语句1
elif 条件2:
   执行语句2
elif 条件3:
   执行语句3
```

- **elif必须和if一起使用，否则出错**

# 三、循环语句

## 1.定义：

```
一般情况下，需要多次重复执行的代码，都可以用循环的方式来完成

循环不是必须要使用的，但是为了提高代码的重复使用率，所以有经验的开发者都会采用循环
```

## 2.while循环的格式

```python
while 条件:
     条件满足时，做的事情1
     条件满足时，做的事情2
     条件满足时，做的事情3

while 条件1:
   条件1满足时，做的事情1
   条件1满足时，做的事情2
   条件1满足时，做的事情3
while 条件2:
    条件2满足时，做的事情1
    条件2满足时，做的事情2
    条件2满足时，做的事情3
```

## 3.控制语句

```
break
	用来结束整个循环
continue
	用来结束本次循环，紧接着执行下一次的循环

注意
    1.只能用在循环中，除此以外不能单独使用
    2.在嵌套循环中，只对最近的一层循环起作用
```

## 4.for循环

### 定义

```
for循环可以遍历任何序列的项目，如一个列表或者一个字符串等
```

### 格式

```python
for 临时变量 in 列表或者字符串等:
    循环满足条件时执行的代码
```
