# **一、可执行文件**

```cmd
.exe（Executable）: Windows可执行文件格式，包含计算机程序的二进制代码，可在Windows系统上运行。

.msi（Microsoft Installer）: Microsoft Windows Installer文件，用于安装和配置Windows应用程序。

.scr（Screen Saver）: 屏幕保护程序文件，包含用于显示屏幕保护的代码和资源。

.dll（Dynamic Link Library）: Windows共享库文件，包含可由多个程序共享的代码和数据。

.rpm（Red Hat Package Manager）: Linux系统上的软件包格式，用于软件的安装、升级和卸载。

.deb（Debian Package）: Debian和Ubuntu Linux系统上的软件包格式，用于软件的管理和安装。

.sys（System）: Windows系统文件，包含设备驱动程序或操作系统的核心组件。

.com（Command）: 早期Windows系统上使用的命令文件格式，包含可执行的命令行程序。

.lib（Library）: C和C++语言的静态库文件，包含编译后的代码和数据。

.o（Object）: 目标文件格式，包含编译后但未链接的二进制代码。

.obj（Object）: 同样是目标文件格式，用于存储编译后但未链接的目标代码。

.a（Archive）: 静态库的归档文件格式，通常在Unix-like系统中使用。

.so（Shared Object）: Unix-like系统上的共享库文件，包含可由多个程序共享的代码和数据。

.elf（Executable and Linkable Format）: 可执行和可链接格式，常见于Unix-like系统。

.ko（Kernel Object）: Linux内核模块文件，用于动态加载和卸载内核模块。

.la（Libtool Archive）: Libtool工具生成的库文件归档格式，用于跨平台编译。

.plo（Perl Object）: Perl语言的目标文件格式，包含编译后的Perl代码。

.out（Output）: 通用输出文件，可指代各种类型的可执行文件或目标文件。

.coff（Common Object File Format）: 一种目标文件格式，用于Windows和Unix-like系统。

.mach-o（Mach Object）: macOS和iOS系统上的可执行文件格式。

.dex（Dalvik Executable）: Android系统上Dalvik虚拟机使用的可执行文件格式。

.ipa（iOS App Archive）: iOS应用程序的安装包格式，包含可执行文件和资源。

.apk（Android Package）: Android应用程序的安装包格式，包含可执行文件和资源。

.ocx（OLE Control Extension）: Windows系统上用于扩展功能的对象链接和嵌入控件。

.oat（ART Ahead-Of-Time Compiled Executable）: Android系统上ART运行时的输出文件，包含预先编译的本机代码。
```



#  二、脚本文件

```cmd
.mht（MIME HTML Archive）: MHT文件是Web文档的归档格式，将HTML、CSS、图像等保存为单个文件，通常用于保存网页快照。

.vbs（VBScript）: VBScript文件，包含Visual Basic Script代码，通常用于Windows环境中的脚本编程。

.js（JavaScript）: JavaScript文件，包含网页中执行的脚本代码，用于增强网页交互性和动态性。

.php（Hypertext Preprocessor）: PHP文件，包含服务器端脚本，用于创建动态网页内容。

.jsp（JavaServer Pages）: JSP文件，包含Java代码，用于创建动态、基于Java的Web页面。

.asp（Active Server Pages）: ASP文件，包含Microsoft的服务器端脚本代码，用于创建动态网页。

.hta（HTML Application）: HTA文件是一种特殊的HTML文件，用于创建独立的Windows应用程序。

.chm（Compiled HTML Help）: CHM文件是一种帮助文档的编译格式，通常用于Windows平台。

.bat（Batch）: Batch文件，包含一系列命令，通过批处理脚本执行一组任务，常用于Windows命令行。

.PowerShell（.ps1）: PowerShell脚本文件，包含Microsoft PowerShell命令，用于自动化任务和系统管理。

.pl（Perl）: Perl脚本文件，包含Perl编程语言的脚本代码，用于文本处理和系统管理。

.sh（Shell Script）: Shell脚本文件，包含命令行脚本代码，用于在Unix-like系统中执行一系列命令。

.jar（Java Archive）: JAR文件是Java的归档文件格式，通常包含Java应用程序、库和资源。

.py（Python）: Python脚本文件，包含Python编程语言的脚本代码，用于多种应用领域。
```

```cmd

.wpt（WPS模板）: WPS模板文件，用于创建新的WPS文档时应用样式和格式。

.ett（WPS表格模板）: WPS表格模板文件，用于创建新的WPS表格文档时应用样式和格式。

.doc（Microsoft Word）: Microsoft Word文档格式，用于存储文本、图形和样式。

.docx（Office Open XML Word）: Microsoft Word的现代文档格式，采用XML标准，支持更丰富的功能。

.dot（Word模板）: Microsoft Word模板文件，用于创建新的Word文档时应用样式和格式。

.docm（Word宏启用文档）: Microsoft Word启用宏的文档格式，支持包含宏代码的文档。

.xls（Microsoft Excel）: Microsoft Excel表格文档格式，用于存储数据、公式和图表。

.xlsx（Office Open XML Excel）: Microsoft Excel的现代表格文档格式，采用XML标准，支持更丰富的功能。

.xlt（Excel模板）: Microsoft Excel模板文件，用于创建新的Excel表格时应用样式和格式。

.xlsm（Excel宏启用工作簿）: Microsoft Excel启用宏的表格文档格式，支持包含宏代码的工作簿。

.ppt（Microsoft PowerPoint）: Microsoft PowerPoint演示文档格式，用于创建幻灯片和演示。

.pps（PowerPoint幻灯片）: Microsoft PowerPoint幻灯片格式，可用于自动播放演示。

.pptx（Office Open XML PowerPoint）: Microsoft PowerPoint的现代演示文档格式，采用XML标准，支持更丰富的功能。

.pptm（PowerPoint宏启用演示文稿）: Microsoft PowerPoint启用宏的演示文档格式，支持包含宏代码的演示。

.pot（PowerPoint模板）: Microsoft PowerPoint模板文件，用于创建新的演示文档时应用样式和格式。

.ofd（Office文件格式）: Office文件格式，是中国国家标准GB/T 33190-2016中规定的一种办公文档格式。

.pdf（Portable Document Format）: PDF文件格式，用于可移植的文档显示和打印。

.eid（Evernote Import File）: Evernote导入文件，用于将文档导入到Evernote应用中。

.eis（Evernote Export File）: Evernote导出文件，用于将Evernote中的笔记和数据导出。

.eip（Evernote Import Package）: Evernote导入包文件，用于一次性导入多个文档到Evernote应用。

.uof（Uniform Office Format）: 统一办公格式，是一种用于办公文档的开放标准格式。

.uot（Uniform Office Text）: 统一办公文本格式，是UOF格式的一部分，用于存储文本文档。

.uos（Uniform Office Spreadsheet）: 统一办公表格格式，是UOF格式的一部分，用于存储表格文档。

.uop（Uniform Office Presentation）: 统一办公演示格式，是UOF格式的一部分，用于存储演示文档。

.odt（Open Document Text）: 开放文档文本格式，是开放文档标准（ODF）中的一部分，用于存储文本文档。

.ods（Open Document Spreadsheet）: 开放文档表格格式，是ODF标准的一部分，用于存储电子表格文档。

.odp（Open Document Presentation）: 开放文档演示格式，是ODF标准的一部分，用于存储演示文档。
```
# 四、图片文件

```cmd
.png（Portable Network Graphics）: PNG是一种无损的位图图像格式，支持透明背景，常用于网络图像和图标。

.jpg（Joint Photographic Experts Group）: JPG是一种有损的图像格式，适用于照片和高色彩图像，具有较高的压缩比。

.jpeg（Joint Photographic Experts Group）: JPEG是JPG图像格式的扩展名，用于存储照片和图形图像。

.bmp（Bitmap）: BMP是一种无损的位图图像格式，每个像素占用较大空间，通常用于Windows平台。

.gif（Graphics Interchange Format）: GIF是一种支持动画和透明度的位图图像格式，常用于简单动画和图标。

.tif（Tagged Image File Format）: TIFF是一种无损的位图图像格式，支持多页文档和高质量图像。

.tiff（Tagged Image File Format）: TIFF是TIFF图像格式的扩展名，用于存储高品质的图像和矢量数据。

.pcx（Personal Computer eXchange）: PCX是一种早期的位图图像格式，适用于DOS和Windows系统。

.wmf（Windows Metafile）: WMF是一种Windows图形元文件格式，支持矢量图形和文本。

.emf（Enhanced Metafile）: EMF是WMF格式的升级版，支持更高的分辨率和更复杂的图形元素。

 .webp: WebP是由Google开发的图像格式，结合了高压缩和高质量，用于网络图像和网页加载。

.jfif (JPEG File Interchange Format): JFIF是JPEG图像的文件格式，常用于存储压缩的数字图像，与JPEG相似，但在存储时可能包含特定的标记信息。
```

#  五、声音文件

```cmd
.wav（Waveform Audio File Format）: WAV是一种无损音频文件格式，常用于存储高质量音频，支持多种采样率和位深度。

.mid（Musical Instrument Digital Interface）: MID是一种音乐文件格式，包含音符、乐器和控制信息，用于控制电子乐器和演奏音乐。

.mp3（MPEG Audio Layer III）: MP3是一种有损音频文件格式，以高压缩率保存音乐文件，常用于音乐存储和传输。

.AAC（Advanced Audio Coding）: AAC是一种高级音频编码格式，提供更高的音质和压缩效率，常用于移动设备和在线流媒体。

.FLAC（Free Lossless Audio Codec）: FLAC是一种无损音频文件格式，提供无损压缩，保留原始音频质量。

.OGG（Ogg Vorbis）: OGG是一种开放的音频格式，使用Vorbis编码，具有高音质和压缩效率，常用于音频流媒体。

.m4a（MPEG-4 Audio）: M4A是一种音频文件格式，通常包含使用AAC编码的音频数据，常用于iTunes和苹果设备。

.aiff（Audio Interchange File Format）: AIFF是一种无损音频文件格式，常用于存储高质量的音频数据。

.wma（Windows Media Audio）: WMA是微软开发的音频文件格式，常用于Windows Media Player中，支持有损和无损压缩。

.amr（Adaptive Multi-Rate）: AMR是一种用于语音编码的文件格式，常用于电话和语音记录设备。
```
#  六、文本/网页文件

```cmd
.txt（Text Document）: TXT是纯文本文件格式，包含简单的ASCII或Unicode字符，通常用于存储文本信息。

.rtf（Rich Text Format）: RTF是一种文本文件格式，支持文本样式、图像和对象，常用于文档处理和交换。

.xml（eXtensible Markup Language）: XML是一种标记语言，用于存储和传输结构化数据，常用于配置文件和数据交换。

.htm（Hypertext Markup Language）: HTM文件是HTML格式的扩展名，用于存储网页的标记语言和结构。

.html（Hypertext Markup Language）: HTML是用于创建网页的标记语言，包含文本、图像和链接等元素。

.mht（MIME HTML Archive）: MHT文件是Web文档的归档格式，将HTML、CSS、图像保存为单个文件。

.mhtml（MIME HTML）: MHTML文件是一种Web文档归档格式，用于保存HTML文档及其相关资源。

.json（JavaScript Object Notation）: JSON是一种轻量级数据交换格式，易于读写，常用于数据序列化和配置文件，基于键值对的结构使其在Web开发和API通信中广泛应用。
```

#  七、压缩包文件

```cmd
.rar（RAR Archive）: RAR是一种高度压缩的归档文件格式，通常用于将多个文件打包并压缩为单个文件。

.zip（ZIP Archive）: ZIP是一种常见的压缩文件格式，支持多个文件和目录的打包，广泛用于文件传输和存档。

.7z（7-Zip Archive）: 7z是7-Zip压缩工具创建的文件格式，提供更高的压缩比和安全性。

.iso（ISO Image）: ISO是一种光盘镜像文件格式，包含完整光盘文件系统的映像，可用于光盘制作和虚拟光驱。

.tar（Tape Archive）: TAR是一种打包文件格式，通常与其他压缩算法（如gzip）结合使用，用于创建归档文件。

.tgz（Tar Gzip）: TGZ是使用Tar打包并使用gzip压缩的文件格式，常见于Unix-like系统中。

.gz（Gzip）: GZ是使用gzip进行单一文件压缩的格式，通常用于压缩文本文件。

.gzip（Gzip）: GZIP是一种压缩算法和文件格式，常与Tar结合使用，用于创建.tar.gz文件。

.lzma（LZMA Archive）: LZMA是一种高压缩算法，LZMA Archive格式用于创建经过LZMA压缩的文件。

.bz2（Bzip2）: BZ2是一种压缩算法和文件格式，常与Tar结合使用，用于创建.tar.bz2文件。
```

#  八、邮件文件

```cmd
.eml（Electronic Mail Message）: EML是标准的电子邮件文件格式，通常由邮件客户端使用，存储邮件内容和附件。

.msg（Outlook Mail Message）: MSG文件是Microsoft Outlook的邮件格式，包含邮件的文本内容、附件和元数据。

.pst（Personal Storage Table）: PST文件是Microsoft Outlook中用于本地邮件存储的文件格式，包含邮件、联系人等。

.mbox（Mailbox）: MBOX是通用的邮件存储格式，多封邮件存储在同一个文件中，常用于UNIX和类Unix系统。

.ost（Offline Storage Table）: OST文件是Microsoft Outlook的离线存储文件，用于在离线模式下访问邮件。

.dbx（Outlook Express Mailbox）: DBX文件是Outlook Express使用的邮件存储格式，包含邮件和文件夹结构。

.emlx（Mail Message）: EMLX文件是用于在Apple Mail中存储单封邮件的文件格式。

.p7m（Cryptographic Message Syntax Standard）: P7M文件是加密的邮件文件，使用Cryptographic Message Syntax标准进行加密和签名。
```
